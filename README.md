# StepGeomChecker

Lightweight STEP / IGES / STL viewer and exporter for Windows.

## Features
- 3D viewing
- Rotate / Zoom / Pan
- Section view
- Basic measurement
- Assembly tree browsing
- Export to STEP / IGES / STL
